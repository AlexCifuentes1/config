📖 Documentación

¿Cómo usar?

Selecciona uno de los reproductores arriba según tus necesidades:

  • LIVE (6s): Para transmisiones en directo con bajo delay
  • Estable: Para streaming con máxima estabilidad
  • Personalizado: Para interfaz personalizada

Características DRM

Los reproductores soportan:

  • Widevine: DRM de Google para streaming protegido
  • ClearKey: Sistema de encriptación abierto

Requisitos

Para usar los reproductores necesitas:

  • Navegador moderno (Chrome, Firefox, Edge, Safari)
  • Soporte para DASH (Dynamic Adaptive Streaming over HTTP)
  • Soporte para EME (Encrypted Media Extensions)
  • Stream DASH con formato .mpd

Ejemplo de URL

Formato esperado para el stream:

https://ejemplo.com/stream/manifest.mpd
Parámetros DRM:
- drmScheme=clearkey
- drmLicense=keyID:key

Configuración Personalizada

Puedes modificar los reproductores editando los parámetros de Shaka Player:

player.configure({
    streaming: {
        bufferingGoal: 60,      // Segundos de buffer
        rebufferingGoal: 10,    // Segundos antes de reproducir
        lowLatencyMode: true    // Modo bajo delay
    }
});

Solución de Problemas

  • No reproduce: Verifica que el stream sea accesible y el DRM sea correcto
  • Se traba: Usa el reproductor "Estable" con más buffer
  • Mucho delay: Usa el reproductor "LIVE" con bajo delay
  • CORS error: El servidor puede estar bloqueando solicitudes cross-origin
